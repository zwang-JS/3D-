# 由Gemini3.0生成的3D粒子捕捉系统.......
这是一个由Gemini3.0生成的3D粒子捕捉系统。目前最推荐的版本： v1.1 & v4.0

功能介绍如下：
首先有三种方式可以控制这对粒子，第一种是鼠标拖动可以调整角度
第二种需要获取摄像头权限，当你的手不在摄像头取景框中时，那么左下角就会显示waiting hand，当我们把手放入取景框时，就会显示hand detected，把手放进来的话，摄像头就可以监测你手的运动来控制这堆粒子
第三种控制方式就是这个律动，当你把这个律动模式打开之后，他就会获取你的麦克风权限，然后根据你麦克风接收到的声音来让这堆粒子振动

右上角的辉光效果就是让粒子一闪一闪，关掉的话也不会影响粒子最基础的变色功能
右下角是取景框
左下角可以显示实时fps也能调节粒子大小

以下为版本介绍:

v1.0:只有可怜的30种模型，但是效果能凑合用😒

v1.1:（扩充版）增加到了100种模型，可玩性大大增加😍

v2.0: (适配版)由于手机浏览器有严格的安全限制，如果你是通过 IP 地址（如 http://192.168.x.x）访问，或者部署在没有 SSL 证书的网站上，浏览器会直接禁止访问摄像头，且不会弹出询问窗口
（温馨提醒：手机配置过低的就不要尝试移动端了，作者的手机是oppo Findx8 Ultra，开性能模式最高也只有90帧，平均稳80帧）❤️❤️❤️

v3.0:这个版本是单独拿出来的，模型最全的还是v1.1版本，此版本修改了一下交互逻辑，在页面的操作台中有小字说明，添加了一些带有浪漫元素的模型(❁´◡`❁)

v4.0:此版本大升级，可以添加照片，之后将照片替换成粒子，把喜欢的人的照片拼成圣诞树一定很浪漫吧😍

v5.0：煮包正在努力赶制中。。。部署到服务器上分享给别人就对小白友好，而且实现保存功能给ta一个惊喜吧😜
